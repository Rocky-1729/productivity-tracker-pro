# 📈 Productivity Tracker Pro

### Track Tasks • Build Habits • Reach Goals

A modern full-stack productivity management platform designed to help users organize tasks, build daily habits, maintain consistency, and improve productivity through analytics, streak tracking, calendars, and goal management.

---

Create a `.env` file in the root directory.

```env
GEMINI_API_KEY=your_gemini_api_key
JWT_SECRET=your_super_secret_key
APP_URL=http://localhost:5173
```

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/Rocky-1729/productivity-tracker-pro.git
```

Navigate to the project:

```bash
cd productivity-tracker-pro
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```
